# Júlia Costa Lash Designer — Catálogo

Catálogo responsivo da Júlia Costa Lash Designer, desenvolvido com **Vite + TypeScript**.

## Páginas

| Página | Descrição |
|---|---|
| `index.html` | Catálogo principal com serviços |
| `stories.html` | Galeria de transformações (antes e depois) |

## Scripts

```bash
npm install        # Instalar dependências
npm run dev        # Servidor de desenvolvimento
npm run build      # Build de produção (saída: dist/)
npm run preview    # Preview do build
npm test           # Rodar testes Playwright
```

## Stack

- [Vite](https://vitejs.dev/) — bundler e dev server
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Playwright](https://playwright.dev/) — testes E2E

## Deploy

O site é publicado via **GitHub Pages** a partir da branch `gh-pages`.
